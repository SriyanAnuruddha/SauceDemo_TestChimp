## Rationale

Engagement step between add-to-cart and checkout. Useful for funnel drop-off analysis between cart and checkout-started.

## Metadata keys

| Key | Values | Notes |
|-----|--------|-------|
| `cart.item_count_bucket` | `1` \| `2-3` \| `4+` | Items in cart when viewed |
| `cart.is_empty` | boolean | Whether cart has zero items |
